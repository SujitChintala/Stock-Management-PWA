# Ice Cream Distributor Management System

A mobile-first Progressive Web App (PWA) for managing ice cream distribution to pushcart sellers.

## Features

* Manage brands, sellers, and items
* Record daily stock distribution
* Automatic discount calculations
* Track payments and remaining balances
* Future support for analytics and ML insights

## Tech Stack

Frontend: Next.js (PWA)
Backend: FastAPI
Database: PostgreSQL

## Architecture

Mobile PWA → FastAPI API → PostgreSQL
